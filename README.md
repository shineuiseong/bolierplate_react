## 🌻 소개

반복되는 기본 기능을 만드는데 숙달이 되었다면, 기본템플릿으로부터 프로젝트를 시작하자 라는 마음으로 보일러 플레이트라는 템플릿을 제작하였습니다.

## 📲 기능

- 메인화면
- 로그인
  - 카카오
  - 구글
- 로그아웃
- 마이페이지
  - 닉네임 수정
  - 회원 탈퇴
